# QA Fundamentals and Manual Testing Portfolio

## Overview

This repository contains my practical training projects in **Manual Testing and QA Fundamentals**, completed during QA coursework and exam preparation.

The projects demonstrate my ability to:

* Design and execute test cases
* Identify and document software defects
* Work with test management templates
* Perform API testing using Postman
* Organize QA documentation in a structured and traceable way

---

## Repository Structure

Each folder represents a separate QA project and contains:

* **Test Cases** – documented functional test scenarios
* **Bug Reports** – identified issues with priority and severity
* **Postman Collection** – API requests used for testing (where applicable)
* **Project README** – short description of the specific project
